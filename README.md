# 我的Django RESTFramework使用文档 


## #1 命名规范 

### #1.1 app

- 所有app必须在项目根目录下
- 命名 : app_+名字
- 新建app后,在app目录下添加urls.py文件

### #1.2 文件/文件夹

- 所有文件/文件夹均已蛇形命名法命名(小写字母+下划线)

### #1.3 类/函数/变量 

- 类 : 大驼峰命名(首字母大写)
- 函数 : 蛇形命名法
- 变量 : 蛇形命名法 
- 全局变量 : 全大写

## #3 我的插件
